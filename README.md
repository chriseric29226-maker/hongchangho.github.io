# 홍창호 포트폴리오 홈페이지

## 바로 확인
`index.html` 파일을 브라우저에서 열면 됩니다.

## 실제 인터넷에 공개하는 방법
### GitHub Pages
1. 새 GitHub 저장소를 만듭니다.
2. 이 폴더의 모든 파일을 업로드합니다.
3. 저장소 Settings → Pages → Deploy from a branch를 선택합니다.
4. `main / root`를 선택하면 공개 주소가 생성됩니다.

### Netlify
Netlify의 배포 화면에 이 폴더 전체를 드래그하면 공개 주소가 생성됩니다.

## 수정 위치
- 본문 내용: `index.html`
- 색상·레이아웃: `style.css`
- 메뉴·문의 폼 동작: `script.js`
- 이미지: `assets/`
